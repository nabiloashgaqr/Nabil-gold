# 📋 Gold AI Signals - شروط إدارة الصفقات
## Trade Management Rules

---

## 1️⃣ شروط الدخول (Entry Rules)

### 1.1 الشروط الأساسية
```
✅_signal = ALL(conditions)

الشروط الأساسية المطلوبة:
├── وقت التداول: ضمن الجلسة (11:00-17:00 UTC, Sun-Thu)
├── جودة الجلسة: HIGH أو أفضل
├── مستوى الثقة: ≥ 70%
├── نسبة المخاطرة/العائد: ≥ 1.5
├── عدد الصفقات المفتوحة: < 3
├── إجمالي السحب: < 10%
└── لا توجد أخبار عالية التأثير قريبة
```

### 1.2 شروط إضافية حسب نوع الإشارة

#### BUY Signal
```
شروط إضافية للشراء:
├── السعر الحالي < EMA 50 (اتجاه صاعد)
├── RSI < 70 (ليس ذروة شراء)
├── MACD_histogram > 0
├──_support_price = find_support(current_price)
└── distance_to_support < 20_points
```

#### SELL Signal
```
شروط إضافية للبيع:
├── السعر الحالي > EMA 50 (اتجاه هابط)
├── RSI > 30 (ليس ذروة بيع)
├── MACD_histogram < 0
├── resistance_price = find_resistance(current_price)
└── distance_to_resistance < 20_points
```

### 1.3 رفض الإشارة (Signal Rejection)
```
⛔_reject_signal_if:
├── news_within_30min = TRUE
├── high_impact_news = TRUE
├── spread > 5_points
├── ATR < 1.0
├── consecutive_losses >= 3
└── daily_signals >= 8
```

## 2️⃣ إدارة وقف الخسارة (Stop Loss Rules)

### 2.1 حساب وقف الخسارة
```
stop_loss_distance = ATR × 1.5

للشراء (BUY):
└── SL = entry_price - stop_loss_distance

للبيع (SELL):
└── SL = entry_price + stop_loss_distance
```

### 2.2 قواعد تحريك وقف الخسارة
```
TRAILING_STOP_RULES:

┌─────────────────────────────────────────────────────────────┐
│ المراحل:                                                     │
│                                                             │
│ 1️⃣ المرحلة الأولى: TP1 Hit (50% من المسافة للهدف)            │
│    ├── إغلاق 50% من الصفقة                                   │
│    ├── تحريك SL إلى نقطة التعادل (Entry Price)              │
│    └── تفعيل التتبع (Trailing)                               │
│                                                             │
│ 2️⃣ المرحلة الثانية: التتبع (Trailing)                        │
│    ├── SL يتحرك للأعلى (BUY) / للأسفل (SELL)                │
│    ├── المسافة: 20 نقطة من السعر الحالي                     │
│    ├── خطوة التحديث: كل 5 نقاط                               │
│    └── لا يتحرك للخلف أبداً                                  │
│                                                             │
│ 3️⃣ المرحلة الثالثة: TP2 Hit (100% من المسافة للهدف)          │
│    ├── إغلاق الـ 50% المتبقية                                │
│    └── SL يتحرك لأقرب 15 نقطة من السعر                      │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 قواعد الإغلاق الجزئي
```
PARTIAL_CLOSE_RULES:

 عند الوصول لـ TP1:
├── إغلاق: 50% من حجم الصفقة
├── إيقاف عن العمل: 50% المتبقية
└── تحريك SL إلى التعادل

 عند الوصول لـ TP2:
├── إغلاق: 100% من الحجم المتبقي
└── الصفقة مكتملة ✅
```

## 3️⃣ أخذ الربح (Take Profit Rules)

### 3.1 حساب الأهداف
```
tp1_distance = ATR × 2.0
tp2_distance = ATR × 3.5

للشراء (BUY):
├── TP1 = entry_price + tp1_distance
└── TP2 = entry_price + tp2_distance

للبيع (SELL):
├── TP1 = entry_price - tp1_distance
└── TP2 = entry_price - tp2_distance
```

### 3.2 قواعد الأهداف المتقدمة
```
ADVANCED_TP_RULES:

┌────────────────────────────────────────────────────────────┐
│ قاعدة التقسيم:                                              │
│                                                            │
│ إذا كانت نسبة الربح المتوقعة > 3× SL:                       │
│ ├── TP1 = عند 2× SL                                        │
│ └── TP2 = عند 3.5× SL                                      │
│                                                            │
│ إذا كانت نسبة الربح المتوقعة < 2× SL:                       │
│ ├── TP1 = عند 1.5× SL (هدف واحد فقط)                       │
│ └── لا TP2                                                 │
└────────────────────────────────────────────────────────────┘
```

## 4️⃣ إدارة المخاطر (Risk Management Rules)

### 4.1 حدود المخاطر
```
RISK_LIMITS:

┌────────────────────────────────────────────────────────────┐
│ المخاطرة لكل صفقة:            1-2% من رأس المال            │
│ المخاطرة اليومية:             6% من رأس المال               │
│ الحد الأقصى للصفقات المفتوحة: 3 صفقات                      │
│ وقف السحب الأقصى:            10% من رأس المال               │
│ الحد الأقصى للخسائر المتتالية: 3 صفقات                     │
└────────────────────────────────────────────────────────────┘
```

### 4.2 قواعد حجم الصفقة
```
POSITION_SIZE_CALCULATION:

risk_amount = account_balance × risk_percent
pip_value = (position_size × point_value)
position_size = risk_amount / (SL_distance × pip_value)

مثال:
├── رأس المال: $10,000
├── المخاطرة: 1%
├── وقف الخسارة: 20 نقطة
├── قيمة النقطة: $0.1 (لـ 0.01 lot)
└── الحجم: $100 / (20 × $0.1) = 0.05 lot
```

### 4.3 قواعد إيقاف التداول
```
TRADING_HALT_RULES:

🚫 إيقاف مؤقت إذا:
├── السحب وصل: 5% → تحذير (تقليل الحجم 50%)
├── السحب وصل: 7.5% → تحذير قوي (صفقات HIGH فقط)
├── السحب وصل: 10% → إيقاف كامل (لا صفقات جديدة)
├── 3 صفقات خاسرة متتالية → مراجعة الاستراتيجية
└── أخبار عالية التأثير → إيقاف 30 دقيقة قبل الخبر

✅ استئناف التداول:
├── السحب انخفض لأقل من: 5%
├── مراجعة الصفقات الخاسرة
└── موافقة مدير المخاطر
```
