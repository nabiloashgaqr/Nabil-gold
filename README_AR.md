# حزمة الرفع النهائية

هذه الحزمة تحتوي على **جميع الملفات المعدلة/المضافة** في المشروع حتى الآن، جاهزة للرفع إلى المستودع.

## ملاحظات
- تم **استبعاد** الملف المؤقت: `tmp_trades.json`
- تم الحفاظ على نفس هيكل المجلدات حتى تستطيع رفعها مباشرة فوق المشروع.
- الحزمة تشمل:
  - كود التنفيذ والتحليل
  - Telegram / Dashboard / API
  - Session planner / pending governance / scenario governance
  - final evaluation / release readiness
  - الاختبارات
  - ملفات SQL / migrations الموجودة
  - workflow / runbook / templates
